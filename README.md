The AIS Planner (AIJ 2026)

This repository will gradually host the source code, benchmarks, and supporting material associated with our AIJ 2026 paper.

The repository is currently under active development. The codebase is being cleaned for public release, and some components may change over time as implementation details are improved and hard-coded cases are removed or generalized.

At the moment, certain parts of the workflow depend on third-party executables and external tools. I am currently reviewing constraints before deciding what can be included directly in this repository.

Where direct upload is not possible, setup instructions and dependency documentation will be provided so the experiments can still be reproduced.
